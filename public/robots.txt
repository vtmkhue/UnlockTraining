- Run docker-compose in the background
docker-compose up -d --force-recreate --build --remove-orphans

- Install PHPCS
+ Install PHP (if doesn't existed - Download from Internet)
+ Install Composer (if doesn't existed - Download from Internet)
+ Install PHPCS by command below in Terminal
composer global require "squizlabs/php_codesniffer"
+ Check code with PHPCS and standard 12 of PSR for folder code/php 
phpcs --standard=PSR12 ./code/php

- Install STAN
+ Install PHP (if doesn't existed - Download from Internet)
+ Install Composer (if doesn't existed - Download from Internet)
+ Install STAN by command below in Terminal
composer require --dev phpstan/phpstan
+ Check code with STAN and standard is 6 for folder code/php 
vendor/bin/phpstan analyse -l 6 /code/php
